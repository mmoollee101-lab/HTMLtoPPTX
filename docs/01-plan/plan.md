# Plan — HTML → 편집가능 PPTX 변환기

## 배경 / 목적
완성된 HTML 발표자료를 **편집 가능한** PPTX(네이티브 텍스트박스/도형)로 변환한다.
받는 사람이 텍스트만 갈아끼워 템플릿으로 재사용하는 것이 목표.

## 핵심 결정 (확정 — 변경 금지)
- 스크린샷/이미지 방식 **금지**. 결과물은 클릭하면 글자가 수정되어야 함.
- 변환 엔진: `dom-to-pptx@1.1.10` (computed style 측정 → 절대좌표 배치 + 폰트 임베드).
- 렌더링: headless Chromium(`puppeteer`)로 HTML을 실제 페이지로 띄운 뒤 변환 실행.
  (dom-to-pptx가 `getComputedStyle`/Canvas 등 브라우저 API에 의존하기 때문.)
- 마스터 슬라이드/플레이스홀더 구조 불필요. 편집가능한 박스 더미로 충분.

## 요구사항
1. CLI: `입력 HTML`, `출력 PPTX(생략 시 자동)`, `슬라이드 셀렉터(기본 .slide)`.
2. 배치 모드: 폴더 입력 시 내부 `.html` 일괄 변환.
3. 셀렉터 미발견 시 명확한 에러(탐색한 셀렉터 명시).
4. 진행 상황/결과 경로 콘솔 출력. 코드는 단순/가독성.
5. 간단한 웹 앱(업로드 → 변환 → 다운로드).

## 비목표 / 수용된 한계
- 편집가능 변환은 100% 픽셀 동일이 아님(PowerPoint 재흐름). 템플릿 용도라 허용.
- 픽셀 완벽 동일이 필요하면 PDF 변환 → 본 프로그램 범위 밖.

## 완료 기준
- 텍스트/제목/불릿/이미지/간단 SVG 차트 포함 2~3장 16:9 샘플 변환 성공.
- 결과 PPTX: (a) 텍스트 편집 가능, (b) 16:9 유지, (c) 줄바꿈이 화면과 크게 다르지 않음.
- 셀렉터 인자/배치 모드/에러 처리 정상 동작.
